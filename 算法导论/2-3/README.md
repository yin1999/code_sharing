逆序对
=====
## 问题描述
    假设A[1...n]是一个有n个不同数的数组。若i<j且A[i]>A[j]，则对偶(i,j)称为A的一个逆序对（inversion）
## 问题
### a
    列出数组（2，3，8，6，1）的5个逆序对。
### b
    由集合{1，2，···，n}中的元素构成的什么数组具有最多的逆序对？它有多少逆序对？
### c
    插入排序的运行时间与输入数组中逆序对的数量之间由什么关系？证明你的回答。
### d
    给出一个确定在 n 个元素的任何排列中逆序对数量的算法，最坏情况需要O(nlgn)时间。    
    （提示：修改归并排序。）
## 解题
二分法 使用归并排序的结构计算逆序数     
归并排序在根目录/排序算法文件夹中
